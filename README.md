# Car Recommendation System

This project recommends cars to users based on different inputs like budget, fuel type, mileage, and car features.

## How it works
- The user gives some details (budget, preferences, fuel type, etc.)
- The system checks the dataset
- It shows the best matching cars

## How to Run
1. Download or clone this project:
git clone https://github.com/garvupadhyay2004/Car-Recommendation-System.git
